# Driver-Drowsiness-Detection System
## Project Overview
Driver drowsiness is one of the leading causes of road accidents worldwide. This project aims to detect signs of driver fatigue using Computer Vision techniques.
The system uses OpenCV-based face and eye detection to monitor the driver's eye state. If the driver's eyes remain closed for a prolonged period, the system generates a drowsiness alert.

## Objectives
* Detect the driver's face in real time.
* Monitor eye activity continuously.
* Identify prolonged eye closure.
* Generate an alert when drowsiness is detected.

## Technologies Used
* Python
* OpenCV
* NumPy
* Google Colab

## Methodology
1. Capture video frames.
2. Detect face using Haar Cascade Classifier.
3. Detect eyes within the face region.
4. Monitor eye closure duration.
5. Display a drowsiness alert if eyes remain closed.
